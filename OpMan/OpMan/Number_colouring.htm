Number colouring

Where you have values in input boxes either in an indicator area or where you are setting a value for a measurement when moving a machine component, you will find different colours for the numbers have different meanings.

The number colouring conventions for input boxes is:

§  Green on a black background where a value is current, for example the component is at that current position.

§  White on red background where a value for an input box is too high.

§  White on dark blue background where a value for an input box is too low.

§  Black on a white background where a value is valid.

§  Yellow on a black background where a value is a target value when you are moving a machine component.

§  White on light blue background where a value for an input box is too low.

§  Black on a teal background where a value  has not yet been committed
