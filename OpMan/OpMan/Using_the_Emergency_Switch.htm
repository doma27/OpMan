Using the Emergency Switch


 The EDT machine involves working with dangerous materials in a potentially hazardous area.

Knowing how to stop the machine using the Emergency stops is vital.


For further safety Health and Safety details refer to the Maintenance manual.

In the event of an emergency, you can stop the machine whatever process you are running by pushing the Emergency stop button at any time (see E-stop for further details).

 
Important Information

 If you press this button by accident, turn the key located in the button clockwise to restart the machine.
