Semi-Auto Sequence

The operation is defined as a sequence of fixed steps. Each step is actioned by the user in sequence (using the buttons on the operator console).

In addition to progressing or reversing through the various stages, numerous parameters can be varied on the fly.

The buttons are illuminated push buttons on the user console. If the buttons are 'available' to be pressed they will flash.  The user interface reflects the current stage in the sequence and therefore what pressing forwards  or backwards  will do.

While you are running in Semi-Auto you can use any of the following buttons to either STOP, PLAY, PAUSE or REWIND the machine as you prove each of the steps that you are running:

NOTE: Before you can start the Semi-Auto sequence, the  button needs to be flashing.

The green button will only flash when all pre-requisites have been met. This is indicated by all icons on the home screen having a green rather than a red background.


 Pressing the STOP button aborts all operations that you are currently running and resets the sequence

 

 

PLAY
Takes the sequence forward a step

PAUSE
Pauses the sequence

REWIND
Rewinds the sequence one step

STOP
Aborts all operations and resets the sequence*

 

*NOTE: The STOP button is always active whichever operation you are performing.
