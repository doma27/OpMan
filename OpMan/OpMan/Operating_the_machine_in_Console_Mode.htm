Operating the machine in Console Mode

To enter Console mode:

§  Press the green hand button  on the operator console. Press the button again to exit console mode.


 Console mode will only exit if all systems on the machine are stationary and inactive.

Movements that are initiated using the pendant are ‘modal’. To initiate a movement, you press one of the modal buttons on the console to select a component of the machine and then use the directional buttons on the pendant to move the component that you have selected.

When you press one modal button, the area that was previously selected is automatically de-selected. The machine starts up in the default Saddle mode.


 All console modes except Texture station modes are modal i.e. a single button represents a single area. The Left-hand texture station could be selected with one servo or all servos selected.

When you are working in console mode you will see the screen changing to a layout relevant to the sub-system selected, for example the Headstock Steady outlined in red in the left-hand panel of the screen below.

The left-hand area of the screen with the white background tells you which area of the EDT machine you have selected. The right-hand area with the light blue background shows you the limits and extents of the selected sub‑system.
