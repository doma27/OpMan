Introduction

During texturing the concentration of debris in the system builds up. When it reaches a certain level the batch of dielectric needs to be filtered.

At this stage the Working tank is changed for a fresh batch of dielectric, which has been made up from clean (new or filtered) oil.

To allow continuous operation of the machine, the system has two feed tanks (labelled ‘Tank 1’ and ‘Tank 2’ in the diagram below); only one of which is working (or ‘on-line’) at any one time. While one tank is ‘on-line’ the other may either be filtered or filled up from scratch with the dielectric from the ‘Clean Dielectric Tank 3’.

The filtering function is broken into a series of simple operations that can be performed in sequence automatically or manually if required.

The filtering operation can be performed in AUTOmatic mode or manual mode. To switch between AUTO and manual modes you use a captive key-switch.

NOTE: You can switch between AUTOmatic and manual filtering modes at any time.
