Program Selection & Auto pre-amble

Only programs that are relevant to the selected roll type will be presented for selection from the list (outlined in red in the figure below).

Make a program selection by tapping one of the items on the list to highlight it.

Enter the relevant data (for example the ‘Roll ID’)

Select the desired station.

On making your selections, tap the GO button on the screen.

The effect of tapping the Go button on the User Interface screen is to:

§  Transform programs for the PLC

§  Load programs to the PLC for validation

§  Indicate the programs that have been loaded to the PLC

If programs have been checked by the PLC and described as valid, then this view progresses to the Auto – execute Position view.

Providing all pre-requisites are met, e.g. fume covers are down, the START button on the User Console will flash indicating that it is ready to be pressed. Press the flashing User Console button and the program will start.
