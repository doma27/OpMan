Texture Stations

An EDT machine can have up to two texture stations arranged either side of the roll. These pass along the roll as the texturing process is carried out.


 

Each Texture Station contains between one and three Servos.

Drop Arms (one per Servo) that carry the electrodes, are connected to the Servos and it is these that perform the texturing.

The Texture Station also contains a bath containing dielectric in which the electrodes are immersed. The bath is fed with dielectric from the dielectric fluid plant.
