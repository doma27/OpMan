Manual activation

The Fire system is made up of a series of pipes throughout the installation that distribute CO2 on activation of the fire safety sensors. These sensors can be found just below the nozzles.

If the automatic system fails then the system can be actuated by a break-glass switch. There is also a manual pull handle for the release of the CO2.

If you activate the system manually and the system fails for any reason, then the CO2 cylinders will need to be refilled.
