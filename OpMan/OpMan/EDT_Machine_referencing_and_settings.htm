EDT Machine referencing and settings

The EDT machine must be in a referenced state before it can be used in any of the operational modes - AUTO orSemi-auto or Load/ Unload.

When the machine is first turned on, thesoftware cannot determine the position of the various sub-systems on the machine. Therefore to set up the machine, the sub-systems must be moved to reference points in order to calibrate the position feedback.

NOTE: These reference points are always the two furthest limit points so that you know how far in practice each sub-system can be moved to.

Thereâ€™s an input to the computer in voltage terms that is proportionate to the physical position of the steady, elevation or Y-gauge on the roll manipulator. The nature of this proportionate relationship is linear. Therefore any change in the voltage has a direct relationship to the physical position of the steady, elevation or Y-gauge that you are moving.

In order to calibrate the machine, you need to read off the physical position of the machine on the ruler along the base of the roll manipulator in the real world and input those real-world values in the calibration section of the software on the User console.


 The calibration modes and procedures are described in the Maintenance manual.
