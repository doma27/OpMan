Operator Console

The operator console is the part of the machine used to control all the other components and constituent areas of the installation.

The operator console is supplied with a touch screen that allows you to:

§  Control different areas of the installation. 

§  View the current status of how the EDT machine is running, for example the extent of limits as you move parts of the machine around.

Indicator lights on the top of the console allow you to view the state of your EDT machine and site from a distance.

 

The lights are coloured and correspond to the following statuses:

WHITE

You are controlling the machine locally in manual mode using the pendant.

RED

There is an error and you will have to stop running the machine

AMBER

There is a warning with the machine, you can continue to run the texturing process but with warnings pending

GREEN

Texturing process and machine are running normally

Below the screen, you will find

§  A Windows-standard keyboard. that allows you to key in values manually and perform diagnostic tests on the software that controls the machine.

§  A panel with push-buttons and diagrams that signify the different areas of the machine.

A Pendant on a lead besside the console that allows you to move the different parts of the machine independently depending which mode light is illuminated on the console. This is known as operating the machine in ‘Console’ mode.

If you prefer, you have the option to plug a conventional mouse into the USB port beside the keyboard so that you can point and click on icons rather than tap them.
