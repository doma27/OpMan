Manual Mode

This is determined by the position of the AUTO / Manual Key-switch shown earlier. For manual operation, the key needs to be inserted and turned clockwise.

NOTE: If operating in manual mode, the key is captive in the key switch. To ensure responsibility is taken for changes made when in manual mode, you can only remove the key when operating in AUTO mode.

If in manual mode, you can start any of the individual operations by pressing the appropriate button.

NOTE: The operation will only start if the lights on the mimic panel are indicating that all the prerequisite conditions for that operation have been met.

For example, you can only start the Transfer operation if:

§  The clean oil tank (T3) has a level above ‘Low’ i.e. contains oil ready to transfer.

§  The destination tank is not full.

§  The transfer pump is not isolated.

If any of these prerequisite conditions have not been met, the light on the push-button will flash quickly for a period of two seconds and a message will be displayed on the Operator Console.

To stop an operation at any point, press the same button again.
