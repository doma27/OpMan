Semi-Auto Prerequisites

Before you can run the Semi-Auto sequence the software runs a check to ensure that all pre-requisites have been met.

If they have not been met, you will see the following screen displayed. The status indicators have a red background if the pre-requisite has not been met.

 The status indicators have the following meanings:

Icon

Meaning

All pre-requisites have been met

Faulty physical abort button  on the console

Dielectric oil is not within its age limit

Roll has not been fine-centred

Fume hood on station is not closed

One of the switches at the headstock end of the roll has not been triggered

There is no roll on the machine

The saddle is not referenced

The spindle covers are not closed

The spindle is not referenced

The system is in error i.e. there are error messages in the status area of the user console that have not been addressed.

The switch at the tailstock end of the roll has not been triggered
